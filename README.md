# 오늘어때
>공유 일기장 앱  
>파이어베이스 실사용을 해보기 위해 진행한 프로젝트  
>kotlin-version은 자바로 만든 프로젝트를 코틀린을 사용해서 변환한 것

>코틀린 (자바8 이상)  
>안드로이드 스튜디오  
>androidx
